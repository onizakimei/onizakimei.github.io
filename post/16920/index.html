✈️翻墙&直充po币教程

1月 11 2021•字数:2.3k
杂七杂八

分享一下个人推荐翻墙的方法，从此上po18胡吃海喝

翻墙

先说下这里介绍的是SSR，和VPN不一样，关于两者区别可以看这里。我个人是因为vpn凉了一个就得重新下载新的vpn很麻烦，不如用可以随时更换机场的SSR十分简单粗暴
简单来说就是VPN侧重安全性，上手简单（直接下载就好），但流量特征明显，容易被上头搞凉凉；SSR侧重翻墙，注重流量混淆，但第一次使用需要设置一下（运气不好机场主也会跑路……）

开头废话

前期步骤繁琐了点，要根据设备下相应的软件然后买适合自己的机场（节点提供商），最后把订阅链接复制到软件上，弄好以后就能随意开关了。另外同一机场基本每个软件都能共用。

  • 大概使用方法：买适合自己的机场→下载软件→把订阅链接复制到软件→设置好后就可以随时用啦！(或者可以跳到后面看使用方法)

下载&使用方法

手机的话下载地址要用浏览器打开！！网页加载不了的人可以用流量试试。

先说说几种代理模式。大家选完节点后记得设置一下代理模式。

  1. 直连模式（Direct）
    所有网站都不会通过代理。
  2. 分流/PAC/规则模式(Rule)
    可根据自定的代理规则智能代理。（推荐）
  3. 全局模式（Global，PROXY）
    所有网站都会通过代理。

国区没有，要到海外区买或者网上找共享账号！
共享账号一定不要从设置登陆！要从APPStore登陆！下完后赶紧退出！否则手机会变砖！

Quantumult（付费）
下载
  1. 在购买的机场那复制订阅链接。
  2. 打开软件，点下面的设置
  3. 点击订阅，点右上角＋号，点击服务器
  4. 点击名称，随意填（推荐填机场名字）。然后点击链接，粘贴机场买的订阅链接
  5. 添加完成后会自动返回页面，在服务器那左滑，点击更新
  6. 返回主页面，点击下面中间的圆圈选择节点
  7. 点击主页最上面的按钮，开始运行
Shadowrocket（付费）
下载
  1. 在购买的机场那复制订阅链接
  2. 打开软件，点右上角＋号
  3. 类型（Type）选择Subscribe，在URL粘贴订阅链接，点击完成
  4. 等更新完后下面会出现一堆节点，选择其中一个
  5. 点击最上面未连接旁的按钮开始运行
Spectre
下载
  1. 在购买的机场那复制订阅链接
  2. 打开客户端首页，点击订阅按钮选择＋复制订阅链接
  3. 复制链接后点击客户端主界面的添加服务器
  4. 点击 切换服务器，订阅右侧的+
  5. 在新弹出的窗口中，粘贴订阅链接，点击完成即可完成添加。
  6. 添加完成后，即可选择一个节点，点击连接开始使用。
  7. 添加完成后，主界面会显示节点信息，点击顶部未连接右侧的按钮即可开始使用代理。
ShadowsocksR
下载
  1. 在购买的机场那复制订阅链接
  2. 打开软件
  3. 点左上角的shadowsocksR，左滑删除默认配置（有的话）
  4. 点右下角的＋，点第一个图标添加\升级SSR订阅，点击添加订阅地址，粘贴机场买的订阅链接
  5. 处理完后删除订阅列表的第一个FreeSSR-public，然后确定并升级
  6. 选择其中一个节点
  7. 在下面的功能设置-路由选择代理模式
  8. 点右上角的图标开始运行
Clash
下载
  1. 在购买的机场那复制订阅链接
  2. 安装后打开首页，点击配置
  3. 点击新配置
  4. 点击URL (从 URL 导入)
  5. 粘贴订阅链接，点击右上角的保存
  6. 下载完成后选中配置文件，然后返回首页点击点此启动开始使用
  7. 第一次启动时会请求 VPN 权限，需要点击确定允许
  8. 如果需要切换节点，请点开第二个代理选项即可。
ShadowsocksR
下载
  1. 在购买的机场那复制订阅链接
  2. 打开软件（若电脑装了NET Framework 4.0就打开4.0后缀那个）
  3. 找到任务栏里的小飞机图标右键，点击服务器订阅，点击SSR服务器订阅设置
    https://ssr.tools/wp-content/uploads/2018-07-17_163552.jpg
  4. 点击Delete删掉默认的链接，点击Add后在网址粘贴机场买的订阅链接，点确定
  5. 右键任务栏的小飞机，点击SSR服务器订阅设置-更新SSR服务器订阅
  6. 右键任务栏的小飞机，点击服务器，选你购买的机场名称，选择节点
  7. 右键任务栏的小飞机，点击代理规则，选择绕过局域网和大陆
  8. 右键任务栏的小飞机，点击系统代理模式，选择PAC模式或全局模式
  9. 可以运行啦
Clash
下载
  1. 在购买的机场那复制订阅链接。
  2. 安装后可以通过双击通知区域的图标打开主界面
  3. 开启系统代理与开机自动启动
  4. 首先打开Profiles配置选择页面，然后进行配置文件的添加
  5. 复制订阅URL，粘贴到Download from a URL输入框内，点击右侧的Download按钮。
  6. 显示Success!表示成功添加了配置文件，并且下面会多了一个配置，点击可以切换到对应的配置。
  7. 打开Proxies代理服务器选择页面。
  8. 点击Global标签，选择 AUTO（自动选择节点）或是直接选择自己想要使用的节点。
  9. 选择节点后可能没有立即生效，建议返回General首页页面点击Clash for Windows字样通过快速重启客户端重新载入配置。
  10. 这时 Clash 已通过设置系统代理的方式进行工作，使用系统代理的软件已可以正常使用。还是不行的话可以重启电脑试试。
ShadowsocksX-NG-R
ShadowsocksX-NG-R
  1. 在购买的机场那复制订阅链接。
  2. 打开软件
  3. 找到任务栏里的小飞机图标右键-服务器订阅-SSR服务器订阅设置
  4. 删掉默认的链接，粘贴机场买的订阅链接
  5. 重新右键小飞机–SSR服务器订阅设置-更新SSR服务器订阅
  6. 重新右键小飞机-服务器-选择节点-可以翻墙啦

直充po币

  1. 首先你要有一张银行储蓄卡
  2. 下载银联的云闪付app
  3. 绑定储蓄卡
  4. 在po18上充值的时候选用银联卡支付，支付的时候会弹出app，然后直接付款就好了
